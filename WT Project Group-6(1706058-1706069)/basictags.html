BASIC HTML TAGS

  • < HTML > :

    • --The tag tells the browser that this is an HTML document.
    --The tag represents the root of an HTML document.
    --The tag is the container for all other HTML elements (except for the tag).

    [ Syntax ]:

    In HTML, the syntax for the tag is:
    < html >
    < /html >

  • < HEAD > :

    • --The element is a container for all the head elements.
    --The element can include a title for the document, scripts, styles, meta information, and more.
    --The following elements can go inside the element:
    < title > (this element is required in an HTML document)
    < style >
    < base >
    < link >
    < meta >
    < script >
    < noscript>

    [ Syntax ]:

    In HTML, the syntax for the tag is:
    < head >
    < /head >

  • < TITLE > :

    • --The < title > tag is required in all HTML documents and it defines the title of the document.
    --The < title > element:
    ~ defines a title in the browser toolbar
    ~ provides a title for the page when it is added to favorites
    ~ displays a title for the page in search-engine results

    [ Syntax ]:

    In HTML, the syntax for the tag is:
    < title >
    < /title >

  • < BODY > :

    • --The < body > tag defines the document's body.
    --The < body > element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

    [ Syntax ]:
    In HTML, the syntax for the tag is:

    < body >
    < /body >
